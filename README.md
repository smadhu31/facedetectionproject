# Face Detection Attendance System

A modern AI-powered Face Detection Attendance System built using Flask, OpenCV, and face_recognition. The application enables secure face registration, real-time face recognition, and automatic attendance marking through a webcam.

## 🚀 Features

- Face Registration
- Real-time Face Detection
- Face Recognition
- Automatic Attendance Marking
- Attendance History
- Modern Responsive UI
- Webcam Integration
- CSV Attendance Export

## 🛠️ Tech Stack

- Python 3.11
- Flask
- OpenCV
- face_recognition
- HTML5
- CSS3
- JavaScript
- Bootstrap

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/smadhu31/facedetectionproject.git
cd facedetectionproject
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows

```bash
venv\Scripts\activate
```

Linux / macOS

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

## ▶️ Run the Project

```bash
python app.py
```

Open your browser:

```
http://127.0.0.1:5000
```
